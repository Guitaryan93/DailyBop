GOAL: Serve a website with a new random music artist each day.

STEPS:
- Check dailybop123@gmail.com emails for any new email signups
    - Add new emails to the DB
    - Remove any users that have decided to unsubscribe too! IMPORTANT
- Run the cronjob to generate a new random artist for the day
    - This includes:
        o Getting the YouTube music data
        o Getting all major streaming platform URLs
        o Updating the DB with the new artist data
- Use the new artist data record in the DB to update the HTML template
- Push the generated HTML page to Neocities using curl command and the Neocities API
- Build an email template using the new DB data
- Running a cronjob to send out emails to all the users
